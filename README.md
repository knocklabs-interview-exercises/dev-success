# Developer Success Exercise

This is the technical exercise for the Knock Developer Success role. There are 3 options in this repo:

1. **Part One: Katas** — Implement 4 coding functions in JavaScript or Python (30 min)
2. **Part Two: Implementing Knock** — Add an in-app notification feed to a Next.js app and document your experience in a friction log (1 hour)
3. **Debugging Exercises** — Debug customer code: a Node.js workflow trigger errors and a React Guides implementation issues (30 min)
